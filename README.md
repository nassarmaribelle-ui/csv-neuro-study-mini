\# CSV Neuro Study Mini



Small Python project demonstrating basic research data handling and statistics.



\## What this project does

\- Loads a CSV dataset

\- Cleans numeric variables and standardizes column names

\- Runs Pearson correlation and a simple linear regression

\- Produces a scatter plot and a text-based stats summary

```powershell

python -m venv .venv

.\\.venv\\Scripts\\activate

pip install -r requirements.txt
